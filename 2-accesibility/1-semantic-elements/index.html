semantic elements = elements with a meaning

provide the user a good way to navigate and interact with your site. make your html semantic. semantics is about using the correct element in html. there are approximately 110 elements in html

two of them have no meaning - div and span. they tell us nothing about the content. they have no built in accesibility features, so we should always check to see if any other elements are better suited. example of semantic elements: form, table, article. they clearly define the content and its purpose.

in this example, we have five elements:

  • a heading
  • a paragraph
  • a button
  • a link
