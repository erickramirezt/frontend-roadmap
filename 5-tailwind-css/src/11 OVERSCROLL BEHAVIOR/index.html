overscroll behavior

utilities for controlling how the browser behaves when reaching the boundary of a scrolling area


preventing parent overscrolling

use overscroll-contain to prevent scrolling in the target area from the triggering scrolling in the parent element, but preserve bounce effects when scrolling past the end of the container in operating systems that support it
