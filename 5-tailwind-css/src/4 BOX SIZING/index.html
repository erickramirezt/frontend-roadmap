box sizing

utilities for controlling how the browser should calculate an element's total size


including borders and padding

use the box-border to set an element's box-sizing to border-box, telling the browser to include the element's border and padding when you give it a height or width. this means a 100px by 100px element with a 2px border and 4px of padding on all sides will be rendered as 100px by 100px, with an internal content area of 88px x 88px

excluding borders and padding

use box-content to set an element's box-sizing to content-box, telling the browser to add borders and padding on top of the element's specified width or height this means a 100px x 100px element with a 2px border and 4px of padding on all sides will actually be rendered as 112px x 112px, with an internal content area of 100px x 100px
