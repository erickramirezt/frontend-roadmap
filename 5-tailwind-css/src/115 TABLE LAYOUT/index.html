table layout

utilities for controlling the table layout algorithm


auto

use table-auto to allow the table to automatically size columns to fit the contents of the cell

fixed

use table-fixed to allow the table to ignore the content and used fixed widths for columns. the width of the first row will set the column widths for the whole table you can manually set the widths for some columns and the rest of the available width will be divided evenly amongst the columns without explicit width
